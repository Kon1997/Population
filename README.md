# Population
Population simulator made in TypeScript.
## Run
With **yarn**
```
yarn && yarn start
```
With **npm**
```
npm install && npm start
```
## Rules
The [rules] of simulation are inspired by [Hopson's Empire], which can be found [here][rules]
## Technology
VanillaJS + TypeScript + Parcel
